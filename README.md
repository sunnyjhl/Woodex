### Run Locally

To run **Woodex** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/woodex.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/woodex.git
```
